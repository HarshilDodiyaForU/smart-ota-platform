## Recommended Screenshot Specifications

- **Format:** PNG or JPG
- **Resolution:** 1920x1080 or higher
- **Aspect Ratio:** 16:9
- **File Size:** Keep under 500KB per image (optimize if needed)

## How to Take Screenshots

1. **Login Page:** Show the login form with placeholder text
2. **Dashboard:** Show the device table with sample data (at least 3-4 devices)
3. **Upload Page:** Show the drag & drop zone with a file selected
4. **OTA Update:** Show the update available modal/dialog
5. **Flutter App:** Show the mobile app running in Chrome or emulator

## Image Optimization

Use tools like:
- [TinyPNG](https://tinypng.com/) for compression
- [Squoosh](https://squoosh.app/) for optimization
- ImageMagick for batch processing
