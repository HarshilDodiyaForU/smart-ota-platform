- Type password: password

## Scene 2: Dashboard (5-15 seconds)
**Visual:** Switch to Device Dashboard tab
**Narration:** "Monitor your entire device fleet in real-time"

**Action:**
- Show device table with multiple devices
- Highlight status badges (Up to Date, Update Available)

**Narration:** "Upload new firmware with drag-and-drop simplicity"

- Drag a .zip file into the drop zone

## Tips for Recording:
1. Use screen recording software (OBS, Loom, or QuickTime)
2. Record at 1080p resolution
3. Use a clean browser window (no bookmarks bar)
4. Pre-populate database with sample devices for dashboard demo
5. Have a sample firmware.zip file ready
6. Keep transitions smooth and quick
7. Add background music (optional, royalty-free)

## Post-Production:
- Add title card at start (2 seconds)
- Add closing card with GitHub link (2 seconds)
- Total video length: ~34 seconds
- Export as MP4, 1080p, 30fps
